# Currency Exchange
## Features
* Get list of currencies to exchange
* Exchange two currencies  
Using external api `https://api.apilayer.com`
## How to use
* create `.env` file  
`DB_USER`  
`DB_PASS`  
`DB_HOST`  
`DB_PORT`  
`DB_NAME`  
`SECRET_KEY` - for jwt  
`EXTERNAL_API_KEY` - api key from https://api.apilayer.com
* use `alembic upgrade init`
